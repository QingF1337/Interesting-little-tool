# 电脑一键优化程序

这是一个基于Python的电脑一键优化程序，可以帮助清理系统垃圾、优化系统性能。

## 功能特点

- ✅ 清理系统临时文件
- ✅ 清理注册表垃圾
- ✅ 管理启动项（支持查看和禁用）
- ✅ 磁盘碎片整理
- ✅ 优化系统服务
- ✅ 直观的图形界面
- ✅ 详细的日志记录
- ✅ 自动请求管理员权限
- ✅ 支持打包为可执行文件

## 环境要求

- Windows 7/8/10/11 操作系统
- Python 3.6 或更高版本
- 管理员权限（运行程序时需要）

## 安装步骤

### 方法一：直接运行（需要Python环境）

1. **安装Python**
   - 访问 https://www.python.org/downloads/ 下载并安装Python 3.6+
   - 安装时选择"Add Python to PATH"

2. **下载程序**
   - 将程序文件下载到本地文件夹

3. **运行程序**
   - 以管理员身份运行命令提示符
   - 导航到程序所在目录
   - 执行命令：`python main.py`

### 方法二：使用可执行文件（无需Python环境）

1. **打包程序**
   - 在有Python环境的电脑上，运行 `build.bat` 脚本
   - 脚本会自动安装PyInstaller并打包程序

2. **使用打包后的文件**
   - 打包完成后，可执行文件位于 `dist\电脑优化大师.exe`
   - 将此文件复制到任何Windows电脑上
   - 以管理员身份运行该文件

## 打包说明

1. **运行打包脚本**
   - 双击运行 `build.bat` 文件
   - 脚本会自动检查Python环境并安装必要的依赖

2. **获取可执行文件**
   - 打包成功后，可执行文件会生成在 `dist` 目录中
   - 文件名为 `电脑优化大师.exe`

3. **分发可执行文件**
   - 可以将生成的可执行文件分享给没有Python环境的朋友
   - 他们只需以管理员身份运行该文件即可

## 使用方法

1. **运行程序**
   - 以管理员身份运行程序
   - 程序会自动检查并请求管理员权限

2. **执行一键优化**
   - 在主界面中，选择要执行的优化选项（默认全选）
   - 点击"开始优化"按钮
   - 等待优化完成，查看优化结果

3. **管理启动项**
   - 点击"管理启动项"按钮
   - 在弹出的窗口中查看所有启动项
   - 选择不需要的启动项，点击"禁用所选"按钮
   - 点击"刷新"按钮更新启动项列表
   - 完成后点击"关闭"按钮

## 注意事项

- 运行程序时需要管理员权限
- 优化过程中可能会暂时占用系统资源
- 建议定期运行优化程序以保持系统性能

## 程序结构

- `main.py` - 程序入口
- `optimizer.py` - 系统优化核心功能
- `gui.py` - 图形用户界面
- `logger.py` - 日志记录功能
- `logs/` - 日志文件存储目录

## 技术说明

- 使用tkinter库创建图形界面
- 使用subprocess调用系统命令
- 使用winreg操作Windows注册表
- 使用多线程避免界面卡顿
- 详细的错误处理和日志记录
